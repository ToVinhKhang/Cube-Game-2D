Hướng dẫn chạy demo game trên Godot Engine

1. Tải Godot Engine trên các hệ điều hành khác nhau tại đây "https://godotengine.org/download/windows"

2. Chọn phiên bản "Godot 3.2.3 - Mono version" và tiến hành tải

3. Giải nén và chạy file "Godot_v3.2.3-stable_mono_win64.exe" để mở Godot

4. Khi Godot đã hiển thị lên, ta chọn "Import" ở thanh menu bên phải sau đó chọn "Browse" 
	-> Vào thư mục "Jumping Cube- Source/Game" chọn project.godot
	-> "Import & Edit"

5. Sau khi import thành công thì Godot sẽ hiển thị lên trình editor.

6. Để chạy game trên editor ta nhấn F5 hoặc nhấn nút "Play the project" ở góc phải trên cùng.
